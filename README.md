
# Web Scraping and EDA Pipeline

## Overview
This project builds an automated Python pipeline that:

- Scrapes data from a website
- Cleans and preprocesses the data
- Performs exploratory data analysis (EDA)
- Generates insights

The entire workflow runs using one command.



## Project Structure

project/
│
├── data/               
├── scrape.py           
├── clean.py            
├── eda.py              
├── main.py            
├── requirements.txt   
└── README.md



## Installation
Install required libraries:

pip install -r requirements.txt



## Run Pipeline
Execute the full workflow:

python main.py


## Tech Stack
- Python
- Selenium
- Pandas
- Matplotlib



## Output
- Scraped dataset
- Cleaned dataset
- EDA insights and visualizations
